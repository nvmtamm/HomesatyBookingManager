# Homestay Booking Manager UI

React + Vite frontend for the Homestay Booking Manager backend project.

## Run

```bash
cd frontend
npm install
npm run dev
```

## What is included

- Dashboard layout with sidebar navigation
- Summary metric cards
- Tours table
- Booking form section
- Booking activity list
- Homestay occupancy overview
- Business rule alerts

## Next step

Replace the mock data in `src/data/mockData.js` with API calls to your Java backend when the backend endpoints are ready.
